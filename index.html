Primary Abilities (Class Abilities)

4 unique class abilities — each can be enhanced with Ability Runes

🛡️ Tank - Taunt

Description: Taunts all enemies for 5 seconds, forcing them to attack the Tank. Grants +10 flat defense + 2% of total defense during taunt.

Mana Cost: 0 (Automatic when available)

Cooldown: Automatic - triggers when all enemies aggro-ed

Rune Scaling: +50% per tier (Tier 1: +10→+15 def, Tier 5: +10→+20 def)

Healer - Heal

Description: Heals an injured ally for 5% of target max HP + (Bonus Mana × 0.4) + 10 flat HP.

Base Effect: 5% max HP + (Bonus Mana × 0.4) + 10

Mana Cost: 0 (Automatic when available)

Cooldown: Automatic - triggers when ally below full HP

Rune Scaling: +15% effectiveness per tier (affects both % and mana scaling)

Mage - Fireball

Description: Blasts all enemies for 80% attack damage each (AOE).

Base Damage: 80% attack damage (AOE)

Mana Cost: 0 (Automatic when available)

Cooldown: Automatic - triggers when multiple enemies present

Rune Scaling: +30% damage per tier (Tier 1: 104%, Tier 5: 200%)

Rogue - Double Strike

Description: Strikes twice, dealing 150% total attack damage.

Base Damage: 150% attack damage

Mana Cost: 0 (Automatic when available)

Cooldown: Automatic - triggers on attacks

Rune Scaling: +30% damage per tier (Tier 1: 195%, Tier 5: 300%)

How Rune Scaling Works

Example - Fireball with Runes:

  • Base (No Rune): 80% attack damage
  • Tier 1 Fireball Rune: 80% + 24% = 104% attack damage
  • Tier 2 Fireball Rune: 80% + 48% = 128% attack damage
  • Tier 3 Fireball Rune: 80% + 72% = 152% attack damage
  • Tier 4 Fireball Rune: 80% + 96% = 176% attack damage
  • Tier 5 Fireball Rune: 80% + 120% = 200% attack damage

See the Runes Section for complete tier-by-tier values for all abilities.

Each Rune Slot node unlocks 1 slot. You can have up to 5 Rune Slots per character.

⚔️ Combat Mechanics

Damage Calculation

Defense Reduction Formula:

Damage Reduction % = Defense ÷ (Defense + 100)

Actual Damage = Base Damage × (1 - Damage Reduction %)

This creates diminishing returns - each point of defense is slightly less effective than the last.

Defense Damage Reduction Damage Taken
0 0% 100%
25 20% 80%
50 33.3% 66.7%
100 50% 50%
200 66.7% 33.3%
400 80% 20%
900 90% 10%

Critical Hits

Critical Hit Damage = Normal Damage × (Crit Damage % ÷ 100)

Example: With 200% Crit Damage, a critical hit deals 2× normal damage.

Dodge

When an attack is dodged, it deals 0 damage and displays "DODGE"

Dodge chance is rolled before damage calculation

Lifesteal

Lifesteal heals you for a percentage of damage dealt

Example: 10% Lifesteal on a 1,000 damage hit heals 100 HP

Regeneration

HP Regen: Restores HP between dungeon rooms

Mana Regen: Restores Mana per second during combat

🌳 Skill Tree System

How to Access: Press K to open the Skill Tree

Skill Points: Earn 1 skill point per character level (shared across all characters)

Each character has their own skill tree - allocate nodes separately for Tank, Healer, Mage, and Rogue

⚠️ CRITICAL: Rune Slot Nodes

YOU MUST UNLOCK RUNE SLOT NODES TO USE ABILITY RUNES!

Rune Slots Required: To equip an Ability Rune (Taunt, Heal, Fireball, Double Strike), you MUST first allocate a Rune Slot Node in your character's skill tree.

Location: Each of the 5 branches has 1 Rune Slot node at the end of the middle path (red keystone-sized node)

Total Available: 5 Rune Slot nodes per character (one per branch)

Stat Runes: Stat Runes (Attack, Defense, Crit, etc.) do NOT require Rune Slot nodes - they can be equipped freely

Skill Tree Structure

Each character has a skill tree with 5 branches radiating from a center start node. Each branch has 3 paths that fork partway through.

Branch Themes

Branch Theme Focus Stats

Branch 2 (Bottom-Right) Speed/Crit Crit Chance, Crit Damage, Attack Speed
Branch 3 (Bottom-Left) Defense/HP HP, Defense, HP Regen
Branch 4 (Top-Left) Magic/Utility Mana, Mana Regen, CDR

🔴 Rune Slot Nodes (Red Keystone)

Cost: 1 skill point each

Location: End of the middle path in each branch (one per branch, 5 total)

Effect: Unlocks 1 Rune Slot - Required to equip Ability Runes!

Important: You can allocate all 5 Rune Slot nodes to have 5 total rune slots per character.

Ability Runes (Taunt/Heal/Fireball/Double Strike) require Rune Slots - you must unlock these nodes to use them!

Stat Runes (Attack, Defense, Crit, HP, Mana, etc.) do NOT require Rune Slots - they can be equipped freely without unlocking any skill tree nodes!

Keystones can be equipped immediately in their own separate equipment slot - no skill tree nodes required!

🟡 Legendary Nodes (Gold)

Cost: 1 skill point each

Location: At the very end of certain branches

Effect: Gain bonus stats per Epic+ item equipped

Skill Tree Path Example

To unlock a Rune Slot for Ability Runes:

  1. Start from center (Start node - free)
  2. Allocate 5 Small nodes (5 skill points)
  3. Allocate 1 Medium fork node (1 skill point)
  4. Allocate 5 Medium nodes on middle path (5 skill points)
  5. Allocate the Rune Slot node at the end (1 skill point)

Total: 12 skill points to unlock your first Rune Slot

Each additional Rune Slot requires a similar path down another branch.

Strategy Tips

  • Prioritize Rune Slots early if you want to use Runes (both Ability and Stat Runes)
  • Keystones don't need unlocking - you can equip them immediately in the Keystone slot
  • Choose branches that match your character's playstyle (Defense for Tank, Magic for Healer/Mage, etc.)
  • Legendary Mastery nodes are very powerful in late game when you have full Epic/Legendary gear
  • You can respec - skill points can be refunded and reallocated
  • Each character has separate trees - build each one according to their role

Pets

Pets provide passive bonuses that scale with their level. Each character has their own pet slot. Pets can be upgraded up to level 20.

How to Get Pets: Pets drop from Boss kills in specific dungeons

Drop Rates:

  • First Pet: 10% chance from any boss kill
  • Additional Pets: 1% chance from any boss kill

Max Level: 20

Note: Pets are themed by dungeon but provide their stat bonus regardless of which dungeon you're in

Pet Mechanics

  • Bonus Formula: Base Value + (Per Level × (Current Level - 1))
  • Stat Application: Bonuses are added directly to character stats when equipped
  • Multiple Pets: Each character can equip one pet at a time
  • Dungeon Themes: Pets are themed by dungeon (where they drop) but work in any dungeon
